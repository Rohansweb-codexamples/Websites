How to Host Your Website on GitHub Pages

The developer way to go live for free.

Step 1: Create a Repository

Log in to your GitHub account and create a new repository. If you name it username.github.io (where username is your actual GitHub username), your site will be available at that root URL.

Step 2: Upload Your Code

Upload your index.html, style.css, and script.js files. Ensure your main file is named exactly index.html so the server knows where to start.

Step 3: Enable Pages

Navigate to Settings > Pages. Under "Branch", select "main" or "master" and click Save.

Step 4: Go Live

GitHub will provide a URL (usually https://username.github.io/repo-name). It might take 1-2 minutes for the site to process, but then you're live!
